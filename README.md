# PyTakeNote #
- A simple CLI notetaking app written in Python 3
- Needed an upgrade from my note.txt file

## Instrunctions
- List all notes:
    ./PyTakeNote.py
- Add a Note:
    ./PyTakeNote.py add
- Edit a Note:
    ./PyTakeNote edit [Note ID]
- Delete a Note:
    ./PyTakeNote delete [Note ID]
