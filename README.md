# PyTakeNote #
- A simple CLI notetaking app written in Python 3
- Needed an upgrade from my note.txt file

## Instrunctions
- List all notes:
    ./pytakenote.py
- Add a Note:
    ./pytakenote.py add
- Edit a Note:
    ./pytakenote.py edit [Note ID]
- Delete a Note:
    ./pytakenote.py delete [Note ID]
