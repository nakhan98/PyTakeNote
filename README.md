# PyTakeNote #
- A simple CLI notetaking app written in Python 3
- Needed an upgrade from my notes.txt file

## Instructions
- List all notes:
    ./pytakenote.py
- Add a note:
    ./pytakenote.py add
- Show a note
    ./pytakenote.py show [Note ID]
- Edit a note:
    ./pytakenote.py edit [Note ID]
- Delete a note:
    ./pytakenote.py del/delete [Note ID]
